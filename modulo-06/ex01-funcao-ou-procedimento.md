# Exercício 1 — Função ou Procedimento? (Conceitual)

### 1. `Str()`

- **Tipo:** Função.
- **Justificativa:** Converte um número em texto e retorna o valor formatado como string.

### 2. `QOut()`

- **Tipo:** Procedimento.
- **Justificativa:** Apenas executa a ação de exibir uma informação na tela sem retornar nenhum valor para o programa.

### 3. `Date()`

- **Tipo:** Função.
- **Justificativa:** Consulta o sistema e retorna a data atual.

### 4. `Len()`

- **Tipo:** Função.
- **Justificativa:** Calcula e retorna a quantidade de elementos de um array ou caracteres de uma string.

### 5. `Upper()`

- **Tipo:** Função.
- **Justificativa:** Recebe um texto e retorna uma nova string com todos os caracteres convertidos para maiúsculas.

### 6. `ClearScreen()`

- **Tipo:** Procedimento.
- **Justificativa:** Executa apenas a ação de limpar a tela do terminal sem devolver nenhum resultado.
